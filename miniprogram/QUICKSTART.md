# 阿汤的小食堂 - 快速启动指南

## 🚀 5分钟上手

### 第一步：安装微信开发者工具

**下载地址：**
- 官网：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
- Windows稳定版：直接下载 .exe 安装包
- 大小：约200MB

**安装步骤：**
1. 双击安装包
2. 同意协议
3. 选择安装路径
4. 完成安装

### 第二步：打开项目

1. **启动微信开发者工具**

2. **扫码登录**
   - 使用微信扫描二维码登录

3. **导入项目**
   - 点击左侧的 "+" 或 "导入项目"
   - 选择项目目录：`D:\SystemCache\CursorWorkSpace\electDishSystem\miniprogram`
   - 项目名称：阿汤的小食堂
   - AppID：选择"测试号"（或输入您的AppID）
   - 点击"导入"

### 第三步：编译运行

1. **自动编译**
   - 项目导入后会自动编译
   - 左侧模拟器会显示小程序界面

2. **查看效果**
   - ✅ 首页：显示菜品列表
   - ✅ 购物车：查看已添加的商品
   - ✅ 订单确认：填写地址提交订单

3. **真机预览**
   - 点击工具栏的"预览"按钮
   - 使用微信扫描二维码
   - 在手机上查看实际效果

## ⚠️ 注意事项

### 关于 TabBar 图标

当前配置中需要以下图标（暂时可以不准备，会显示默认图标）：
```
images/home.png
images/home-active.png
images/cart.png
images/cart-active.png
```

**临时方案：** 如果没有图标，小程序仍可正常运行，只是TabBar会显示默认图标。

**准备图标：**
1. 访问 https://www.iconfont.cn/
2. 搜索 "home" 和 "cart"
3. 下载 PNG 格式，尺寸 40x40 或 81x81
4. 保存到 `miniprogram/images/` 目录

### 关于 AppID

**使用测试号：**
- 导入项目时选择"测试号"
- 功能完全可用
- 无需注册

**使用正式 AppID：**
1. 访问 https://mp.weixin.qq.com/
2. 注册小程序账号
3. 获取 AppID
4. 在项目中配置 AppID

## 🐛 常见问题

### Q1: 提示 "未找到入口app.json"
**解决：** 确保导入的是 `miniprogram` 目录，不是 `electDishSystem` 根目录。

### Q2: 模拟器显示空白
**解决：** 
1. 检查控制台是否有错误
2. 点击"编译"按钮重新编译
3. 尝试切换到"真机调试"模式

### Q3: TabBar 不显示图标
**解决：** 
- 方案1：准备图标文件（见上方说明）
- 方案2：暂时忽略，不影响功能

### Q4: 真机预览显示"该页面不存在"
**解决：** 
1. 检查 app.json 中的 pages 配置
2. 确保所有页面文件都存在
3. 重新编译项目

### Q5: 提交订单后没有反应
**解决：** 
- 检查是否填写了地址和电话
- 查看控制台是否有错误信息
- 电话号码格式：11位手机号（1开头）
