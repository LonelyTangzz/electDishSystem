# 清理微信小程序缓存

如果修改代码后还是报同样的错误，说明是缓存问题。

## 🔧 解决方法

### 方法一：清理缓存并重新编译（最简单）

**在微信开发者工具中：**

1. 点击菜单栏 **工具 → 清除缓存**
2. 选择：
   - ✅ 清除文件缓存
   - ✅ 清除数据缓存
   - ✅ 清除授权数据
3. 点击 **清除**
4. 然后点击 **编译** 按钮重新编译

---

### 方法二：删除缓存文件（如果方法一不行）

**关闭微信开发者工具后：**

1. 打开文件管理器
2. 进入缓存目录：
   ```
   C:\Users\MrTang\AppData\Local\微信开发者工具\User Data\
   ```
3. 删除以下文件夹：
   - `Default\Cache`
   - `Default\Code Cache`
   - `Default\GPUCache`
4. 重新打开微信开发者工具
5. 打开项目并编译

---

### 方法三：完全重启（终极方法）

1. **关闭微信开发者工具**
2. **任务管理器中结束进程**
   - 按 `Ctrl+Shift+Esc` 打开任务管理器
   - 找到 "微信开发者工具" 相关进程
   - 右键 → 结束任务
3. **删除项目缓存**
   ```
   删除项目目录下的 node_modules（如果有）
   ```
4. **重新打开**
   - 启动微信开发者工具
   - 打开项目
   - 点击编译

---

## 🎯 快速操作步骤

### 最快的方法（推荐）：

```
1. 工具 → 清除缓存 → 全选 → 清除
2. 点击 "编译" 按钮
3. 完成！
```

---

## ✅ 验证是否成功

清理后重新编译，如果看到：
- ✅ 模拟器正常显示页面
- ✅ 底部 TabBar 可以切换
- ✅ 没有报错信息

说明成功了！

---

## 📱 如果还是报错

如果清理缓存后还是报错，可能是：

1. **代码没有保存**
   - 确保所有修改的文件都已保存（Ctrl+S）

2. **文件被占用**
   - 关闭所有相关程序
   - 重启电脑

3. **版本问题**
   - 更新微信开发者工具到最新版本

---

```
Windows:

macOS:
~/Library/Application Support/微信开发者工具/
```
