# 图标目录

此目录存放小程序所需的图标和图片。

## 需要的图标文件

请将以下图标文件放入此目录：

### TabBar 图标（需要准备）
- `home.png` - 首页图标（未选中状态）40x40
- `home-active.png` - 首页图标（选中状态）40x40
- `cart.png` - 购物车图标（未选中状态）40x40
- `cart-active.png` - 购物车图标（选中状态）40x40

### 菜品占位图
- `dishes/dish-placeholder.png` - 菜品占位图

## 图标规范

- 格式：PNG（支持透明）
- TabBar 图标尺寸：40x40 像素或 81x81 像素
- 文件大小：<40KB
- 颜色：未选中状态建议灰色，选中状态使用主题色#FF6B35

## 临时方案

如果暂时没有图标，小程序仍可运行，但TabBar会显示默认图标。

您可以：
1. 使用在线图标生成工具（如 iconfont.cn）
2. 使用Emoji表情替代
3. 使用简单的纯色图标

## 获取图标的方法

### 方法1：iconfont（推荐）
1. 访问 https://www.iconfont.cn/
2. 搜索"home"、"cart"等关键词
3. 下载PNG格式，调整为40x40或81x81

### 方法2：在线生成
可以使用CSS生成简单图标，或使用小程序自带的icon组件

### 方法3：UI设计工具
- Figma
- Sketch
- Adobe XD

制作建议的主题色：#FF6B35
